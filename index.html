Simplify YaCy first launch

Sources : this branch was forked from yacy main repository

This is an attempt to simplify YaCy first launch.

Try now

(make sure you have at least Java 1.7 installed)

  • Launch YaCy with Java Web Start
  • or use YaCy runnable jar :
    • double-click the downloaded file
    • or open a terminal and type : java -jar yacycore-1.83-9802.jar

Why

YaCy main website (http://yacy.net) currently only provide following deployment options :

  • Windows installer
  • Debian package
  • Mac Os application
  • Cross-platform tar.gz archive with .bat and .sh launch scripts
  • Cross-platform sources to compile

I think some other packaging options could be useful and may help anyone who whould like to try running a local YaCy peer. These options could be :

  • Self-contained portable application (with embedded jre)
  • Cross-platform :
    • Runnable jar
    • Java Web Start link
    • Browser applet
    • War package
