Upgrading YaCy

Various ways lets you upgrade your YaCy peer to a more recent version than the latest stable release available on https://yacy.net/.

  • use your peer /ConfigUpdate_p.html administration page
    • , and install a dev prefixed version
  • manually download a tar.gz archive at http://kaskelix.de/update/ or at https://github.com/luccioman/yacy_search_server/releases (these repositories are used by the ConfigUpdate_p.html page)
  • use a Debian repository at http://debian.yacy.net/ (but with a deprecated signing key size) or at https://bintray.com/luccioman/yacy_search_server/yacy
  • use a Docker image (daily built from latest sources) from Docker Hub at https://hub.docker.com/r/luccioman/yacy/
  • clone the main GitHub repository or download the latest sources zip and build your own binaries (see instructions in the Readme file )
